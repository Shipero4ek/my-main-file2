GIT flow

GIT clone-для того, щоб "склонити" репозиторій
GIT pull - для того, щоб отримати зміни з репозиторіями
GIT checkout -b branchename- створити нову гілку та переключитися на неї
GIT.add -додати зміни, як зроблені локально
GIT comit-m"commit text"підписати їх
GIT push origin branchename-запушити на git сервер
